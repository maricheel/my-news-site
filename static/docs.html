msnow.click — Maintenance Guide

Everything you need to run, fix, and extend this website — even if you've never seen it before.

🐍 Python 3.11 ⚛️ React 18 ▲ Vercel 🐘 PostgreSQL 📰 topnewsshow.com

1 What this website is

msnow.click is a news video archive site. It shows daily political TV shows (Morning Joe, Rachel Maddow, Chris Hayes, etc.) as video cards. Videos are automatically pulled from a WordPress site (topnewsshow.com) that embeds Rumble video players.

🆓

Free users

Watch ~20% of daily shows (stable all day, changes at midnight)

💳

Premium users

Pay $9.99/month — admin approves after payment via WhatsApp or email

🤖

Automation

WordPress posts sync to the site every 10 minutes via cron-job.org

6 How Deployment Works

✅ Every git push to the main branch automatically deploys to https://msnow.click in ~60–90 seconds.

How Vercel routes work

The vercel.json sends all routes to api/index.py. Flask handles everything — pages, API, static files.

{
  "builds": [{ "src": "api/index.py", "use": "@vercel/python" }],
  "routes": [{ "src": "/(.*)", "dest": "api/index.py" }]
}

Checking deployment status

Go to vercel.com → your project → Deployments tab.

Checking function error logs

Vercel dashboard → Functions tab → click any invocation to see the full Python traceback.

9 Automation — WordPress → Site Sync

How it works

cron-job.org
every 10 min → POST /api/cron?key=API_KEY → topnewsshow.com
WP REST API → Neon PostgreSQL
  1. cron-job.org calls the cron endpoint every 10 minutes
  2. Fetches 10 latest posts from topnewsshow.com/wp-json/wp/v2/posts?per_page=10&order=desc&_embed
  3. Checks each post's wp_post_id directly in the DB (fast, no HTTP round-trips)
  4. New posts are parsed: title, Rumble embed URL, thumbnail, show name
  5. Inserted into the posts table — oldest first to keep timeline order

Show detection keywords

If no keyword matches, the post is skipped. Add new keywords in api/index.py → _CRON_SHOW_KEYWORDS.

Morning Joe Chris Jansing Reports Katy Tur Reports Deadline: White House The Beat With Ari Melber The Weeknight All In with Chris Hayes The Rachel Maddow Show The Briefing with Jen Psaki The Last Word with Lawrence O'Donnell The 11th Hour with Stephanie Ruhle Velshi Alex Witt Reports PoliticsNation with Al Sharpton The Weekend

How the free 20% selection works

🔢 A date-seeded LCG shuffle selects 20% of videos. Seed = YYYYMMDD integer. Same seed all day = same free videos. Changes at midnight. Refreshing the page does not change which shows are free. 
// In index.html — useFreeIds()
const t = new Date();
let s = t.getFullYear() * 10000 + (t.getMonth() + 1) * 100 + t.getDate();
// LCG shuffle using seed s...
const count = Math.max(1, Math.ceil(ids.length * 0.2));
return new Set(ids.slice(0, count));  // stable all day

14 Common Problems & Fixes

🔄 Site shows old content after a code push
Vercel deployment takes ~90 seconds. Wait, then hard-refresh with Ctrl+Shift+R (or Cmd+Shift+R on Mac).
📺 New WordPress posts not appearing on the site
  • Check cron-job.org → last execution status should be 200
  • Check the response body: {"skipped": 10} means posts exist but no known show matched
  • Go to topnewsshow.com and check the post titles — the show keyword might not be in _CRON_SHOW_KEYWORDS
  • Add the missing keyword (see section 15)
🔐 Admin login stopped working
  • Check Google OAuth credentials are still valid at console.cloud.google.com
  • Make sure GOOGLE_CLIENT_ID env var in Vercel matches your OAuth client ID
  • Make sure you're logging in with hicham1elbanaji@gmail.com
  • Check that msnow.click is in the OAuth authorized domains list
💥 500 errors on Vercel
Go to Vercel dashboard → Functions tab → click the failed invocation. You'll see the full Python traceback. Most common cause: missing or wrong environment variable.
🗄️ Database errors after adding a new column
On Postgres, always use ALTER TABLE ... ADD COLUMN IF NOT EXISTS. Using plain ALTER TABLE when the column already exists aborts the whole transaction. 
-- Correct:
ALTER TABLE users ADD COLUMN IF NOT EXISTS new_col TEXT DEFAULT '';
-- Wrong (will crash if column exists):
ALTER TABLE users ADD COLUMN new_col TEXT;
🔑 Users can't log in — password rejected
Passwords are hashed with Werkzeug's generate_password_hash. The password_plain column is just for admin display — do not try to use it to manually set a password. Use the admin panel's "Create account" feature to reset access.
🎬 Video player doesn't play
Videos are Rumble embeds. If Rumble is down, nothing to fix on our side. Check the rumble_link column in the DB for that post — if it's empty, the WordPress post had no Rumble embed.
🆓 Free 20% selection seems to change unexpectedly
It's seeded by today's date (YYYYMMDD) — same result all day. If it seems to change: check if new videos were added (they change the pool), or if the device clock jumped to a new day.
